- ````ls:```` listado de archivos de una carpeta

- ````rm[ARCHIVO]:```` remove
- ````rmdir[CARPETA]:```` eliminar archivos
- ````mkdir[CARPETA]:```` crear carpeta
- ````touch [ARCHIVO]:```` nombre del archivo
